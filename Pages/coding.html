Simulating the Lotka-Volterra Predator-Prey Model with LaTeX

The Lotka-Volterra model is a fundamental concept in mathematical biology, describing the interaction between predator and prey populations. This model uses two differential equations to represent how the populations evolve over time.

The Model

The equations governing the dynamics are:

Prey population (x):

\[ \frac{dx}{dt} = \alpha x - \beta xy \] where \(\alpha\) is the prey birth rate, and \(\beta\) is the predation rate.

Predator population (y):

\[ \frac{dy}{dt} = \delta xy - \gamma y \] where \(\delta\) is the predator reproduction rate, and \(\gamma\) is the predator death rate.

Numerical Method: Euler's Method

To simulate these equations, we use Euler's method, a simple and widely-used numerical scheme. Euler's method approximates the solution by taking small steps along the curve, updating the population values incrementally:

\[ x_{n+1} = x_n + h \cdot f(x_n, y_n) \]

\[ y_{n+1} = y_n + h \cdot g(x_n, y_n) \]

Where \(h\) is the step size, and \(f\) and \(g\) are the functions representing the rate of change for prey and predator populations.

Visualization

The results of the simulation are plotted using LaTeX's pgfplots package. This plot effectively shows the cyclical interaction between the prey and predator populations over time, reflecting the periodic nature of the Lotka-Volterra model.

Result

Tex Code

                    
\documentclass{standalone}
\usepackage{filecontents}
\usepackage{float}
\usepackage{pgffor}
\usepackage{pgfplots}
\pgfplotsset{compat=1.17}

% Define the Lotka-Volterra equations in the preamble
% Prey equation: dx/dt = \alpha*x - \beta*x*y
\newcommand{\fx}[2]{\alpha*#1 - \beta*#1*#2} 
% Predator equation: dy/dt = \delta*x*y - \gamma*y
\newcommand{\fy}[2]{\delta*#1*#2 - \gamma*#2} 

\newwrite\datfile

\AtBeginDocument{%
    % Parameters for the Lotka-Volterra model
    \def\alpha{0.1} % Prey birth rate
    \def\beta{0.02} % Predation rate
    \def\gamma{0.1} % Predator death rate
    \def\delta{0.01} % Predator reproduction rate
    
    % Initial conditions
    \def\xinitial{40} % Initial prey population
    \def\yinitial{9} % Initial predator population
    \def\h{0.1} % Step size
    \def\nsteps{5000} % Number of steps
    
    % Open the data.dat file for appending
    \immediate\openout\datfile=dataLV.dat
    
    % Write the initial values
    \def\x{\xinitial}
    \def\y{\yinitial}
    \immediate\write\datfile{0\space \x\space \y} % Write time, initial x, initial y
    
    % Perform Euler's method and store the values
    \foreach \i in {1, 2, ..., \nsteps} {%
        \pgfmathsetmacro{\xnext}{\x + \h * (\fx{\x}{\y})}
        \pgfmathsetmacro{\ynext}{\y + \h * (\fy{\x}{\y})}
        \pgfmathsetmacro{\t}{\i * \h}
        \immediate\write\datfile{\t\space \xnext\space \ynext}
        \xdef\x{\xnext}
        \xdef\y{\ynext}
    }
    
    % Close the data.dat file
    \immediate\closeout\datfile
}

\begin{document}
    
    % Plotting the prey and predator populations over time
        \begin{tikzpicture}
            \begin{axis}[
                title={Lotka-Volterra Model: Prey and Predator Populations over Time},
                xlabel={Time ($t$)},
                ylabel={Population},
                grid=both,
                width=12cm,
                height=8cm,
                legend pos=north east
                ]
                \addplot table [x index=0, y index=1, col sep=space] {dataLV.dat};
                \addplot table [x index=0, y index=2, col sep=space] {dataLV.dat};
                \legend{Prey ($x$), Predator ($y$)}
            \end{axis}
        \end{tikzpicture}
    
\end{document}
